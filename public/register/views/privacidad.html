Aviso de Privacidad

Última actualización: 2024

1. Finalidad de este aviso

En Escalablitas / Home-Connect nos tomamos en serio la protección de tus datos personales. Este Aviso de Privacidad explica de forma sencilla qué información recopilamos, cómo la usamos y qué opciones tienes sobre su tratamiento, en el contexto de este proyecto académico.

2. Datos que recopilamos

Al utilizar la plataforma, podemos recopilar los siguientes datos:

  • Datos de registro: nombre completo, correo electrónico, contraseña y número de teléfono.
  • Datos de uso: acciones básicas dentro de la plataforma (por ejemplo, creación de cuenta o actualización de perfil).
  • Datos de perfil profesional: tipo de servicios, experiencia y zona de trabajo (cuando completes tu perfil como profesional).
3. ¿Para qué usamos tus datos?

Utilizamos tus datos personales para:

  • Crear y gestionar tu cuenta en la plataforma.
  • Permitir la conexión entre usuarios y profesionales.
  • Recordar tus preferencias básicas dentro del sistema.
  • Mejorar el funcionamiento general del proyecto académico.
4. Base del tratamiento

El tratamiento de tus datos se realiza con base en tu consentimiento, el cual otorgas al crear tu cuenta y utilizar la plataforma. Puedes dejar de usar la plataforma en cualquier momento y solicitar la eliminación de tus datos de las bases de prueba de este proyecto.

5. Compartición de datos

En el contexto de este proyecto académico, tus datos no se comparten con terceros con fines comerciales. Sin embargo, algunos datos pueden ser visibles para otros usuarios dentro de la plataforma, por ejemplo:

  • Nombre y zona de trabajo de un profesional, cuando este complete su perfil.
  • Información básica necesaria para que un usuario pueda contactar a un profesional.
6. Conservación de los datos

Los datos se conservarán durante el tiempo que el proyecto académico esté activo o mientras mantengas tu cuenta. Posteriormente, los datos podrán ser eliminados o anonimizados para futuros ejercicios académicos.

7. Seguridad de la información

Este proyecto no pretende ser una plataforma comercial en producción, pero aun así se aplican medidas básicas de seguridad para proteger tu información contra accesos no autorizados. Sin embargo, como cualquier sistema en pruebas, no podemos garantizar una seguridad absoluta.

8. Cookies y tecnologías similares

La plataforma puede usar cookies técnicas o de sesión para mantener tu inicio de sesión activo o recordar algunas preferencias básicas. No se utilizan cookies con fines publicitarios en este proyecto académico.

9. Tus derechos

De forma similar a otros marcos de protección de datos, puedes solicitar:

  • Acceder a los datos que has proporcionado.
  • Corregir información incorrecta o desactualizada.
  • Eliminar tu cuenta y los datos asociados a ella dentro del proyecto.

Para ejercer estos derechos dentro del contexto académico, puedes contactar al responsable del proyecto o al equipo desarrollador.

10. Cambios en el Aviso de Privacidad

Este Aviso de Privacidad puede modificarse para reflejar mejoras en la plataforma o cambios en el alcance del proyecto. La versión vigente estará siempre disponible en esta página.

11. Contacto

Para dudas relacionadas con este Aviso de Privacidad dentro del proyecto académico, puedes escribir a: soporte@homeconnect.com (correo de ejemplo para fines académicos).
